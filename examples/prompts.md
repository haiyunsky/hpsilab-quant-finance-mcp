# Example Prompts for Stock and Options Research

Use these prompts in an MCP-compatible client after connecting the HPSILab server.

## Options Research

```text
Analyze NVDA options positioning. Use IV radar, option pressure, max pain, gamma wall, expected move, and squeeze target. Summarize bullish and bearish evidence, then include the key risks.
```

```text
Compare SPY and QQQ for this week's options setup. Focus on IV rank, expected move, risk reversal, max pain, and pressure zones. Do not give investment advice; frame the result as research.
```

## Quant Research

```text
Run a quantitative research view for AAPL. Combine AI prediction, implied volatility, option pressure, Monte Carlo simulation, and equity curve backtests. Return a concise bull/base/bear summary.
```

```text
Use Monte Carlo simulation and backtested equity curves for TSLA. Show the 30-day probabilistic range, downside probability, and which strategy backtests look strongest.
```

## Research Report

```text
Generate a markdown research report for MSFT using all available HPSILab signals. Include AI prediction, volatility analysis, options positioning, Monte Carlo outlook, strategy backtests, and a disclaimer.
```

## Chart Workflow

```text
Generate stock research charts for NVDA and explain what each chart is useful for: price chart, IV surface, and options flow heatmap.
```
